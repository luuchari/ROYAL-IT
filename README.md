# Royal-IT Website

## Overview

Royal-IT is a multi-service business website that combines technology services, beauty services, and community support through the Royal-IT Foundation. The website provides information about the services offered and allows users to make bookings online.

The project is built using **HTML, CSS, and JavaScript** and is designed to be simple, responsive, and easy to expand in the future.

---

## Features

### Navigation

* Responsive navigation bar with hamburger menu
* Works on desktop and mobile devices
* Smooth layout for multiple pages

### Pages Included

* **Home Page** – Introduction to Royal-IT
* **Internet Café Page** – Information about internet and technology services
* **Beauty Bar Page** – Beauty services offered
* **Foundation Page** – Community support and mission
* **Booking Page** – Users can book services online
* **Admin Login Page** – Admin access for managing bookings

### Booking System

* Online booking form
* Service selection
* Date input
* Contact information
* Booking success popup confirmation

### Foundation Section

* Mission, Vision, and Goals
* Large image sections
* Social media links for community engagement

### Footer

* Quick navigation links
* Contact information
* Social media icons (Facebook, Instagram, WhatsApp)

---

## Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and responsive design
* **JavaScript** – Interactivity (navbar menu and booking popup)
* **Font Awesome** – Social media icons
